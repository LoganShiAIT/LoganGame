# 熵之平衡 | Entropic Equilibrium

这是一个关于秩序、混乱与平衡的哲学微型游戏。

## 核心机制
- **建立秩序 (Build Order)**: 增加你的积分，但会提高系统的“熵值”。
- **系统熵值 (Entropy)**: 随着熵值的增高，界面会产生视觉上的“坍塌”（抖动、色散、噪点）。
- **寻求平衡 (Balance)**: 通过牺牲已建立的秩序（降低积分）来换取系统的稳定性。
- **意识碎片**: 随着你的操作，系统会浮现出一些关于存在与逻辑的哲学片段。

## 运行方式
直接在浏览器中打开 `index.html` 即可开始。

## 思想内涵
这个游戏旨在反思现代系统中对“极致有序”的追求。我们习惯于构建复杂的逻辑和庞大的架构，却往往忽略了维持这些秩序所消耗的外部资源和产生的“热寂”代价。当系统最终走向不可逆的崩溃时，我们唯一的选择是学会舍弃。
